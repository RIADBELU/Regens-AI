## ✨ Overview

Whether you're a developer building the next big thing, or someone looking for a fast and reliable AI assistant — Regen's AI provides the tools and community support you need to succeed.

Built with a **desktop-first experience**, a clean modern UI, and packed with features that go far beyond basic chat.

## 🚀 Core Features

### 💬 AI Chat
- ChatGPT-style chat interface with animated bubble responses
- Powered by **Regen's AI 1.0** — our own proprietary language model
- Streaming responses with smooth typing animations
- Markdown formatting support inside chat
- Auto-generated conversation summaries with key takeaways & action items

### 🎙️ Voice I/O
- **Push-to-talk** voice input for hands-free interaction
- **Click-to-play** audio playback for AI responses
- Natural, responsive voice experience

### 🖼️ Image Generation
- AI-powered image generation with multiple style options
- Styles include: Cinematic, Ultra-realistic, Retro Neon, Illustration, Anime, Fantasy
- Powered by **Regen's Nordic 2.0** — our own proprietary image model
- Customizable watermark settings + instant download

### 🎬 AI Video Generation
- Generate cinematic AI videos with customizable duration, FPS, and motion smoothing
- Powered by **Regen's Voxel 1.2**

### 🌍 Translation
- Real-time translation across **52 languages**
- Powered by **LibreTranslate** (no external API key required)

### 🔍 Web Search Summaries
- Smart web search with summarized results
- Powered by **Brave Search**

### 🔐 Login & Security
- Full authentication system with persistent sessions
- Sign in via: Email, Phone, Google, Meta, Apple, Samsung, or Guest
- **Two-Factor Authentication** (TOTP-based with backup codes)
- Real-time login monitoring with IP geolocation & device detection
- AI-powered suspicious activity detection

### 🗣️ Community Forum
- Full-featured discussion platform
- Create topics, share insights, ask questions, collaborate

### 📝 Blog Platform
- Integrated blogging system for the community
- Write articles, tutorials, and updates

### 🔔 Smart Notifications
- Customizable email alerts for security events, newsletters, and account activity
